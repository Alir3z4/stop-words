Stop Words
==========

List of common stop words in various languages.



Available languages
-------------------
* Arabic
* Danish
* Dutch
* English
* Finnish
* French
* German
* Hungarian
* Italian
* Norwegian
* Portuguese
* Romanian
* Russian
* Spanish
* Swedish
* Turkish

Basic usage
-----------------
```python
from stop_words import get_stop_words

stop_words = get_stop_words('english')
```
